## AI Email Processing System

Automation workflow for processing incoming emails using Gemini AI.
The system watches Gmail inboxes, analyzes email content with AI, stores structured data in Google Sheets, and generates automated responses or notifications.

### Workflow Overview
Gmail Watch → Gemini AI Processing → Google Sheets Logging → Gmail Draft Creation → LINE Notification

### Features
- Gmail monitoring
- AI email analysis
- Auto categorization
- Save structured data to Google Sheets
- Draft email generation
- LINE notification support

### Tools
- Gmail API
- Google Gemini AI
- Google Sheets
- LINE Messaging API
- Make.com
