#### AI Workflow Error Detection System

ระบบตรวจจับข้อผิดพลาดของ Workflow แบบอัตโนมัติ โดยใช้ Google Gemini AI วิเคราะห์ Error และส่งแจ้งเตือนผ่าน LINE Notify เพื่อช่วยตรวจสอบปัญหาได้รวดเร็วขึ้น

### Features

ตรวจจับ Workflow Error อัตโนมัติ
วิเคราะห์ Error Message ด้วย Gemini AI
ส่งแจ้งเตือนเข้า LINE แบบ Real-time
แยกประเภท Error เพื่อให้อ่านง่าย
ใช้สำหรับ Monitoring ระบบ Automation

### Workflow Overview
RSS / Workflow Error → Google Gemini AI Analysis → LINE Alert Notification

### Tech Stack
Make.com
Google Gemini AI
LINE Messaging API
AI Error Analysis
Automation Workflow

### Use Case
เหมาะสำหรับระบบ Automation ที่ต้องการแจ้งเตือนเมื่อ Workflow ล้มเหลว หรือ API มีปัญหา เพื่อช่วยลดเวลาการ Debug ระบบ
