## AI Stock News Aggregator

Automated AI-powered stock news monitoring system.
This workflow collects stock news from multiple RSS feeds, combines the data, analyzes it using Gemini AI, and sends summarized investment insights directly to LINE.

###Workflow Overview
RSS Feeds → Array Aggregator → Gemini AI Analysis → LINE Notification

### Features
- Multi-source RSS aggregation
- AI-generated stock summaries
- Risk and insight analysis
- Automated LINE delivery

### Tools
- Make.com
- RSS Feed
- Google Gemini AI
- LINE Messaging API
- Array Aggregator

### Example Output
- Stock summary
- AI analysis
- Risk factors
- Investment insights
